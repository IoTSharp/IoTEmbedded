## 0. 范围

- 本仓库只承载 MCU 上的 **BASIC 解释器内核** 与最小的 **BasicRuntime 接口适配**（GPIO/UART/I2C/SPI 等）。
- 不承载云端代码生成器、签名服务、计费等任何上层逻辑；那些位于 `IoTSharp.SaaS`。

## 1. 里程碑

| 里程碑 | 状态 | 描述 |
| --- | --- | --- |
| M0 | ⬜ | 解释器最小可运行（Token / Lex / Parse / Eval） |
| M1 | ⬜ | BasicRuntime 接口注册表 v1（与 C# / C 版对齐的最小子集） |
| M2 | ⬜ | 脚本加载、签名校验与本地版本管理 |
| M3 | ⬜ | 一个真实设备（STM32F4 或 STM32G4）上跑通端到端示例 |
| M4 | ⬜ | 资源优化：代码体积、堆栈、ROM 占用 |

## 7. 接口稳定性公约

- **BasicRuntime 接口签名**：与 `external/Edge`（C#）、`external/Pixiu`（Linux C）的同名接口保持等价语义；新增接口需先在三方一致。
- **脚本签名格式**：完全跟随 SaaS 端定义；本仓库只校验，不定义。
- 不在本仓库内引入任何租户 / 计费 / License 概念。
